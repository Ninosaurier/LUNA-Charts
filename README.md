## How to install
1. First clone the repo: 
```
git clone https://github.com/One-Type-Man/accessible-charts.git
```
2. Open the terminal and switch in the project folder: 
``` 
cd accessible-charts
```
3. Install the packages: 
```
npm install 
```
4. Start Storybook: 
``` 
npm run storybook
```
5. Open Chrome and call:
``` 
localhost:8080
```
